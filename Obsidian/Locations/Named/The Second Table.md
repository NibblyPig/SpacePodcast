# The Second Table

Location: Epsilon Ring — Civic Band

The Second Table is a narrow noodle bar set into the corridor wall, marked only by a modest sign and a serving counter recessed behind a glass panel.

Alongside it runs a single long table, extending the length of the unit, with seating on both sides made possible by the wider span of the corridor at this point.

The table surface gleams under the overhead lighting, polished to a uniform smoothness that reflects light cleanly along its length.
